## 🔧 安装步骤（Chrome）

### 第 1 步：解压插件包
将下载的 `xhs-format-helper-v1.0.0.zip` 解压到一个**不会删除的文件夹**（例如桌面上的 `xhs-plugin/` 文件夹）。

> ⚠️ 解压后不要删掉这个文件夹，插件运行时需要读取里面的文件。

### 第 2 步：打开扩展管理页
在 Chrome 地址栏输入：
```
chrome://extensions
```
按回车打开。

### 第 3 步：开启开发者模式
在页面右上角，找到 **「开发者模式」** 开关，点击打开。

![开启开发者模式](https://developer.chrome.com/static/images/get_started/extension-page.png)
*开启后会出现工具栏*

### 第 4 步：加载插件
1. 点击左上角 **「加载已解压的扩展程序」**
2. 选择刚才解压的文件夹（里面要有 `manifest.json`）
3. 点击「选择文件夹」

### 第 5 步：确认安装成功
在扩展列表中能看到 **「小红书排版助手」** 的卡片，右上角蓝色开关已打开 ✅

## 🔧 安装步骤（Edge 浏览器）

### 第 1 步：解压插件包（同上）
### 第 2 步：打开扩展管理页
在 Edge 地址栏输入：
```
edge://extensions
```
### 第 3 步：开启开发者模式
页面左下角打开 **「开发人员模式」**
### 第 4 步：加载插件
点击 **「加载解压缩的扩展」** → 选择解压文件夹

## ❓ 常见问题

**Q：插件工具栏没有出现？**
A：请确保你在 `creator.xiaohongshu.com` 的笔记编辑页面，刷新页面试试。

**Q：解压文件夹后能删掉吗？**
A：**不能**。插件运行时需要读取这些文件，删除后插件会失效。建议放在桌面或文档文件夹。

**Q：如何更新插件？**
A：下载新版本 → 解压覆盖旧文件夹 → 回到 `chrome://extensions` 页面 → 点击插件卡片上的 🔄 刷新按钮。

**Q：激活码丢了怎么办？**
A：联系客服（微信/邮箱），提供购买订单号即可补发。
